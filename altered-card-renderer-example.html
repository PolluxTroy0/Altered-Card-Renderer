Altered Card Renderer
A standalone JavaScript renderer for Altered TCG cards. Embed it on any page with a single <script> tag — no build step, no framework, no dependencies. Fetches card data from the API and draws the full card into an HTML canvas.
Live demo
Embed on your page
Via jsDelivr — no hosting required
Load directly from CDN. Drop the script tag anywhere, then use <altered-card> wherever you want.
<script src="altered-card-renderer-minified.js"></script>

<altered-card ref="ALT_CORE_B_AX_07_C"></altered-card>
<altered-card ref="ALT_CORE_B_AX_07_U_8623" locale="fr"></altered-card>
zero hosting
Self-hosted
Drop altered-card-renderer.js on any static host. The card API and image CDN both expose Access-Control-Allow-Origin: * — no server-side code needed.
<script src="/path/to/altered-card-renderer-minified.js"></script>
